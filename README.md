ToolBox 是一个多功能平台，旨在托管丰富的插件生态系统，将强大的工具无缝集成到您的日常工作中。作为该生态系统的扩展，**OCR识别插件**为您提供了一个高效的文字识别工具，支持从图片中提取文字内容。

## 什么是 OCR 识别插件？

OCR识别插件是 ToolBox 的文字识别工具，基于 RapidOCR 引擎，支持从图片中精准提取文字。插件提供直观的用户界面，支持图片上传、剪贴板粘贴识别、识别结果编辑、历史记录管理等功能。

## 主要功能

- **图片上传：** 支持点击上传或拖拽图片到识别区域。
- **剪贴板粘贴：** 自动检测剪贴板中的图片，一键识别。
- **精准识别：** 基于 RapidOCR 引擎，支持多种语言文字识别。
- **结果编辑：** 识别结果支持实时编辑修改。
- **文字块选择：** 点击识别结果中的文字块，在图片上高亮显示对应区域。
- **历史记录：** 自动保存识别历史，支持快速查看和清除。
- **服务管理：**
  - 启动/停止 Python OCR 服务
  - 查看服务状态和详情
  - 运行诊断检测环境问题
  - 自动安装 Python 依赖
- **设置面板：** 配置服务端口、超时时间、Python 解释器路径等参数。
- **暗色模式：** 完整支持亮色/暗色主题切换。

## 配置与使用

1. **安装：** 从 ToolBox 插件商店安装 `OCR识别` 插件。
2. **启动服务：** 首次使用需点击"启动服务"按钮启动 Python OCR 服务。
3. **上传图片：** 点击上传区域选择图片文件，或直接拖拽图片到识别区域。
4. **粘贴识别：** 复制图片到剪贴板后，插件自动检测并提示识别。
5. **查看结果：** 识别完成后，结果显示在下方文本区域，可直接编辑。
6. **选择文字块：** 点击文本中的文字块，图片上会高亮显示对应区域。
7. **管理历史：** 点击"历史记录"按钮查看过往识别记录。
8. **配置设置：** 切换到"设置"标签页，配置服务端口、超时时间等参数。
